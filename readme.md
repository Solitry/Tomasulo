## 运行方式

​    所有运行方式需要使用 java 1.8版本。

### 1. 直接运行已编译版本

​    bin文件夹下已经给出编译完成的版本，可以直接使用如下指令进行运行：

```
cd bin
java tomasulodisplay.TomasuloDisplay
```

### 2. 编译运行

​    使用如下指令进行编译运行：

```
cd src
javac */*.java
java tomasulodisplay.TomasuloDisplay 
```
### 3. Eclipse工程运行

   提交文件夹为Eclipse项目工程，直接打开项目进行运行即可。
